# 活动效果分析平台 - 产品需求文档

## 1. 产品概述

活动效果分析平台是一款面向市场团队的专业数据分析工具，专为复盘发布会、展会和线上直播活动而设计。平台通过可视化数据展示、多维度分析和智能洞察，帮助市场团队全面评估活动效果，优化未来活动策略。

- **目标用户**：市场团队、活动策划人员、营销管理者
- **核心价值**：提升活动ROI、优化资源配置、沉淀活动经验、支持数据驱动决策

## 2. 核心功能

### 2.1 用户角色

| 角色 | 权限说明 |
|------|----------|
| 管理员 | 配置活动、管理权限、生成报告、分享报告 |
| 分析师 | 查看所有数据、生成报告、添加复盘备注 |
| 查看者 | 查看已分享的报告和基础数据 |

### 2.2 功能模块

1. **活动概览页**：活动基础信息、核心指标卡片、趋势图表、目标达成进度
2. **渠道分析页**：渠道来源对比、成本效益分析、渠道转化漏斗
3. **转化漏斗页**：全流程转化漏斗、环节转化率、流失节点识别
4. **参会人明细页**：参会人列表、多维度筛选、人群画像分析
5. **报告中心页**：报告生成、报告管理、分享功能

## 3. 核心流程

### 3.1 活动复盘流程

用户创建活动 → 配置活动目标和预算 → 导入/录入活动数据 → 查看各维度分析 → 识别问题和亮点 → 添加复盘备注 → 生成报告 → 分享给团队

```mermaid
flowchart TD
    "创建活动" --> "配置目标与预算"
    "配置目标与预算" --> "导入/录入数据"
    "导入/录入数据" --> "活动概览分析"
    "活动概览分析" --> "渠道效果分析"
    "渠道效果分析" --> "转化漏斗分析"
    "转化漏斗分析" --> "参会人明细查看"
    "参会人明细查看" --> "识别问题与亮点"
    "识别问题与亮点" --> "添加复盘备注"
    "添加复盘备注" --> "生成图文报告"
    "生成图文报告" --> "分享给团队"
```

### 3.2 数据分析流程

选择分析维度 → 查看可视化图表 → 对比不同维度数据 → 发现异常/亮点 → 深入下钻分析 → 得出结论

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深蓝渐变 (#1e3a5f → #2d5a87) 配合亮橙色强调色 (#ff6b35)，体现专业数据分析氛围
- **辅助色**：青色 (#00d4aa) 用于正向指标，红色 (#ff4757) 用于警示/流失
- **背景风格**：深色主题，卡片采用半透明玻璃拟态效果
- **按钮风格**：圆角胶囊按钮，悬停时发光效果
- **字体**：标题使用 "Plus Jakarta Sans"，正文使用 "Inter"，数字使用等宽字体
- **布局风格**：左侧导航栏 + 顶部面包屑 + 内容区卡片式布局
- **图表风格**：渐变填充、圆角柱状图、动态动画

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 活动概览 | 活动配置面板 | 卡片式布局，表单输入，下拉选择器 |

### 4.3 响应式设计

- **桌面优先**：主要针对1440px及以上屏幕优化
- **平板适配**：导航栏可折叠，卡片自适应排列
- **移动端**：底部导航，卡片堆叠，图表简化

### 4.4 交互动效

- **页面切换**：淡入淡出 + 轻微上移
- **卡片悬停**：轻微上浮 + 阴影增强
- **数据加载**：骨架屏 + 数字滚动动画
- **图表动画**：柱状图逐条生长，折线图路径绘制
- **漏斗动画**：从上到下依次展开

## 5. 数据指标定义

| 指标名称 | 定义 | 计算方式 |
|----------|------|----------|
| 报名数 | 完成报名的总人数 | 去重计数 |
| 签到数 | 实际到场/上线的人数 | 去重计数 |
| 观看时长 | 平均观看时长 | 总时长/观看人数 |
| 互动次数 | 点赞、评论、提问总数 | 求和 |
| 留资数 | 提交联系信息的人数 | 去重计数 |
| 成交数 | 产生购买行为的人数 | 去重计数 |
| 转化率 | 各环节转化比例 | 下一环节数/当前环节数 |
| ROI | 投资回报率 | (成交金额-成本)/成本 |
| CPA | 单个获客成本 | 总成本/留资数 |
